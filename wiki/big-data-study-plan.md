## Overview
**Target total:** ~300 hours | **Plan total:** ~285–310 hours
**Format:** 11 modules → resources → 5 integration projects → week-by-week sequence → gap flags

## Module Map

- **Module 1 — Generative AI for Big Data and Analytics (~20h)**
  LLM fundamentals, prompt techniques, GenAI platforms, data-use cases, limitations and Responsible AI.
- **Module 2 — Generative AI Systems on Big Data Platforms (~35h)**
  RAG architecture, embeddings and vector search, Spark/Kafka integration, HDFS/S3 data lakes, LLMOps, end-to-end capstone.
- **Module 3 — Introduction to Big Data (~10h)**
  Definition/characteristics of Big Data, evolution of data processing, scalable/fault-tolerant architecture, role of cloud, edge computing & AI.
- **Module 4 — Fundamentals of Big Data (~20h)**
  Big Data use cases, intro to public cloud (AWS, Azure), Apache Hadoop & Spark overview, HDFS storage, YARN.
- **Module 5 — Big Data Ingestion (~30h)**
  Apache Sqoop, Apache Flume, Spark Streaming, Apache Flink, Apache NiFi pipelines, Apache Kafka, producers/consumers.
- **Module 6 — Big Data Storage (~30h)**
  HDFS vs S3, NoSQL fundamentals, Apache HBase, Apache Cassandra, MongoDB.
- **Module 7 — Big Data Analytics (~35h)**
  SQL table creation/querying at scale, Apache Pig, Apache Hive, Impala, Lambda vs Kappa streaming architectures.
- **Module 8 — Big Data Processing with Apache Spark (~45h)**
  Spark introduction, Core/DataFrame APIs, DStream/Structured Streaming APIs, Spark applications, distributed processing model.
- **Module 9 — Big Data Modeling and AI (~40h)**
  ML fundamentals, ML on public cloud, Apache Spark ML/MLlib, pipeline creation, classification/regression, clustering/collaborative filtering.
- **Module 10 — Data Visualization (~15h)**
  Visualization fundamentals, Apache Hue, Jupyter notebooks, Power BI.
- **Module 11 — Security and Access Control (~15h)**
  Authentication/authorization/encryption basics, Kerberos for cluster access, Apache Ranger/Atlas for ACLs.

## Project Milestones
1. Stand up a single-node Hadoop cluster and run a YARN job.
2. Build an end-to-end ingestion pipeline (Kafka/NiFi → storage) and model data in multiple NoSQL systems.
3. Compare Hive/Pig analytics with a Spark DataFrame rewrite.
4. Complete a Spark MLlib pipeline (classification + clustering).
5. **Capstone:** End-to-end system (Ingest → Store → Spark ML → RAG/LLM interface → Visualize).
