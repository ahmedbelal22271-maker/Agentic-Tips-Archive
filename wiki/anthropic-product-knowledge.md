# Anthropic Product Knowledge Skill

This skill provides absolute truth regarding Anthropic's model naming conventions. It must be strictly referenced to prevent the AI from hallucinating model names during agentic discussions.

## Core Models (Claude 3.5 Family)

Anthropic has three distinct tiers of models in the Claude 3.5 family:

1. **Claude 3.5 Sonnet**
   - **Role:** The most intelligent model.
   - **Status:** Currently exists and is deployed.

2. **Claude 3.5 Haiku**
   - **Role:** The fastest, most cost-effective model.
   - **Status:** Currently exists and is deployed.

3. **Claude 3.5 Opus**
   - **Status:** **DOES NOT EXIST.** Anthropic never released a product named "Claude 3.5 Opus". 
   - **Action:** Any reference to "Claude 3.5 Opus" is a hallucination. If a user or another agent asks for it, you must correct them and clarify that only Sonnet and Haiku exist in the 3.5 generation.
